What is TypeScript?
An open-source language created by Microsoft
Static type-checker and ES2015/16/17 transpiler
First class support for types, classes and modules
A superset of JavaScript that compiles to plain JavaScript

Why TypeScript?
Makes code easier to analyse.
Reliable Refactoring.
Better IDE Support.
Can catch errors early.
Incrementally move an existing JavaScript code base to TypeScript.

/**
 * Takes a string and adds "padding" to the left.
 * If 'padding' is a string, then 'padding' is appended
 * to the left side.
 * If 'padding' is a number, then that number of spaces
 * is added to the left side.
 */
function padLeft(value: string, padding: any) {
    if (typeof padding === "number") {
        return Array(padding + 1).join(" ") + value;
    }

    if (typeof padding === "string") {
        return padding + value;
    }

    throw new Error(`Expected string or number, got '${padding}'.`);
}
